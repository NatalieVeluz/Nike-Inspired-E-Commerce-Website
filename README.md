# Nike-Inspired E-Commerce Website

## Overview

The **Nike-Inspired E-Commerce Website** is a front-end web application built using **HTML5**, **CSS3**, and **JavaScript**. Inspired by the design and shopping experience of Nike's official online store, the website allows users to browse products, explore different categories, manage a shopping cart, and navigate through multiple pages while demonstrating the fundamentals of front-end web development.

The website features a clean and modern interface, consistent navigation, responsive layouts, animations, and client-side interactivity to provide an engaging shopping experience. As a front-end project, it does not include a backend server or database, with all functionality handled on the client side using JavaScript.

---

## Features

- Multi-page website with consistent navigation
- Modern user interface inspired by Nike's e-commerce website
- Home page with featured products
- About Us page
- Product catalog with category navigation
- Shopping cart functionality using JavaScript
- Product reviews section
- Contact Us page
- Embedded Google Map
- User registration interface
- Responsive layout and navigation
- Animations and interactive user interface
- Client-side dynamic functionality

## Limitations

This project is a **front-end only** e-commerce website and does not include a backend server or database. Product information, shopping cart interactions, and user interface updates are handled entirely on the client side using JavaScript.
